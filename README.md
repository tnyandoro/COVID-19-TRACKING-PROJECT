### Prerequisites

A Morden Browser

### Setup
- Go to [https://github.com/tnyandoro/JavaScript-Capstone]
- Clone the project to your local machine
- Cd into your directory
- Run npm install
- Type: npm run build
- Run npm start

### Setup
- To Test npm install jest and run npm test
